# Image Selector App

AI生成画像をグループ単位で選別するMacアプリです。

## 機能
- ファイル名から自動グルーピング
- 1グループ最大3枚選択
- 進捗可視化
- JSON保存

## 必要環境
- macOS 13+
- Xcode 15+

## 起動方法

1. Xcodeを起動
2. 新規プロジェクト作成
   - App
   - Language: Swift
   - Interface: SwiftUI
3. 本リポジトリのコードを配置
4. Runボタンで起動

## 使い方

1. 「フォルダ選択」をクリック
2. 画像フォルダを選択
3. 左からグループ選択
4. 画像クリックで選択
